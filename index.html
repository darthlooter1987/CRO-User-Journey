v1.1
April 2026
Archived
🖥 Mock-up
  • Contextual flow recommendations: Screen 2 now adapts based on which landing page was selected — each funnel stage gets a different recommended experiment type with full educational rationale
  • Awareness → A/B Testing recommended (cold traffic + high bounce = validate the Core Hook, not spread thin across combos)
  • Differentiation → MVT recommended (warm audience comparing evidence; Shrinkage borrows signal across related combos)
  • Conversion → A/B + Multi-Armed Bandit recommended (revenue protection — Thompson Sampling auto-shifts traffic to the winner immediately)
  • Loyalty → MVT + Fractional Factorial recommended (smaller high-value audience; CRM-powered priors for fast, accurate results)
  • Recommendation banner with colored badges at top of Screen 2 — changes per funnel stage
  • Deep-dive context panels below experiment cards: "Why this flow?", business size specifics (Small/New vs Big/Old), and "What to Test" tag badges
  • Recommended card gets ★ ribbon; non-recommended card shown as dashed/dimmed (still clickable)

Current

  • Initial version — first complete mock-up of the CRO 3rd Party Setup flow.
  • 3-screen flow: Integrations Hub → Guided Setup Wizard → Connection Detail
  • Integrations Hub with 8 tools across 4 categories (Analytics, Ad Pixels, Email/CRM, Behavior & Heatmaps) — summary strip with connection health metrics

  • 3 connection states: Connected (green), Not Connected (gray), Needs Attention (orange/warning)
