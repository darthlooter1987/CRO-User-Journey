🖥 Mock-up 📄 PRD
  • Carousel showcase: 4 cards now arranged in a horizontally scrollable carousel with left/right navigation arrows and dot indicators, replacing the 2x2 grid

  • Real mock-up image previews: inline HTML mini-renders replaced with high-fidelity landing page screenshots (awareness, differentiation, conversion, loyalty mock-up PNGs)

  • Image hover effect: subtle scale-up (1.03x) on preview images for tactile interaction feedback
  • Scroll-snap alignment for precise card positioning when scrolling or using navigation

April 2026
Archived
🖥 Mock-up 📄 PRD (Superseded)
  • Showcase Dashboard: 4 large cards with rich visual previews of each generated landing page, replacing the old horizontal funnel strip
  • Staggered card reveal animation — confetti burst now fires only on 10/10 completed pages (e.g. Awareness card), rewarding achievement instead of generic animation
  • Enhanced 4-step guided tour with frosted-glass overlay and highlighted action callouts per page
  • Hover tooltips on each card explaining the funnel stage purpose (matching v1.0 educational approach)
  • Readiness score badges with inline progress bars, plus "Finalize now" action buttons per card
  • Awareness page fully completed (10/10) with testimonial & brand story — card styled as "ready to publish" with green border and Publish Now button
  • Distinct info area background on cards for better visual separation from the page preview
  • Checklist "Add now" actions redesigned as high-contrast filled buttons instead of text links
  • Funnel stage labels on entry screen: "Page 1/2/3/4" replaced with Awareness / Differentiation / Conversion / Loyalty, plus "Best for:" campaign lists in colored boxes
  • Card info area backgrounds now use funnel-stage colors (blue, orange, green, purple) for better readability and visual distinction
  • Product images (A1.webp & A2.webp) integrated across all four landing pages — hero shots, comparison visuals, and product detail sections
  • Readiness score color scale: ring and progress bar now use a dynamic red → amber → green gradient based on score (0/10 = red, 5/10 = amber, 10/10 = green)
  • Hover tooltips on showcase cards now display above other cards (z-index fix)
