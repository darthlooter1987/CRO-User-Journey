1. Product Overview

The CRO 3rd Party Setup is a guided integration experience for the Revmatics landing page platform. It enables small business owners — particularly Shopify merchants — to connect their existing analytics tools, advertising pixels, email platforms, and behavior tracking services to their CRO-generated landing pages.

The product is designed around education and simplicity. The target user may not understand why they need a Meta Pixel or Google Analytics connected to their landing pages. The setup flow must teach them the value of each integration before asking them to configure it.

Design Philosophy: The interface follows the Revmatics design language — spacious, clean, premium. Each integration connection follows a guided wizard pattern: learn why → connect credentials → configure pages → verify it works. No dead ends, no unexplained jargon.

The product consists of three core screens:

  • Integrations Hub — portfolio view of all available and connected integrations
  • Guided Setup Wizard — step-by-step connection flow for each tool
  • Connection Detail — management and monitoring for connected integrations

2. Primary User Story

"As a small business owner with a Shopify account, I need to connect my advertising and analytics tools to my landing pages so that I can track which campaigns drive real results — but I need the setup process to explain what each tool does and why I need it, because I'm not a technical marketer."

4. Product Flow Overview

User opens the Integrations Hub from the sidebar navigation
2
Summary strip shows integration health at a glance (total, connected, not connected, needs attention)
3
Integration cards grouped by category show status and quick actions
4
User clicks "Connect" on a not-connected integration to enter the Guided Setup Wizard
5
Step 1 (Learn): Plain-language explanation of what the tool does and why the user needs it

Step 2 (Connect): Enter credentials with clear instructions on where to find them

Step 3 (Configure): Select which landing pages to track and map conversion events

5. Screen Specifications & User Stories

US-01

Integrations Hub — Portfolio View

"As a marketer, I want to see all my available integrations in one place so I can quickly identify what's connected, what's missing, and what needs my attention."
Acceptance Criteria
  • Sticky top navigation with Revmatics branding matching CRO Reporting layout
  • Left sidebar with "Integrations" as active nav item
  • Page title "Integrations" with descriptive subtitle
  • Summary strip: Total Integrations, Connected, Not Connected, Needs Attention, Pages Tracked
  • Integration cards grouped by category (Analytics, Pixels, Email/CRM, Behavior)
  • Each card shows: icon, name, status badge (Connected/Not Connected/Needs Attention), description, connection metadata or action button
  • Connected cards link to Connection Detail screen
  • Not-connected cards show "Connect" button linking to Setup Wizard
  • Needs-attention cards show warning message and "Reconnect" button
  • Design matches existing Revmatics product aesthetic
Mock-up: cro-3rd-party-setup-v1.0-screen1.html
US-02

Guided Setup Wizard — Education Step

"As a small business owner who isn't a technical marketer, I need the setup process to explain what each integration does and why I need it, before asking me to configure anything."
Acceptance Criteria
  • 4-step progress indicator: Learn → Connect → Configure → Verify
  • Step 1 shows plain-language explanation of the tool
  • "Why you need this" section with 3 benefit cards
  • Language is jargon-free and written for Shopify store owners
  • "Skip this step" option for experienced users
  • Progress bar is navigable (can click back to completed steps)
Mock-up: cro-3rd-party-setup-v1.0-screen2.html
US-03

Guided Setup Wizard — Connect & Configure Steps

"As a store owner, I want clear instructions on where to find my tracking IDs and how to enter them, with helpful guidance if I get stuck."
Acceptance Criteria
  • Input fields with clear labels and placeholder examples
  • "Where do I find this?" expandable help section with step-by-step instructions
  • Page selection with toggle switches for each landing page
  • Smart defaults (paid pages ON by default for ad pixels, all pages ON for analytics)
  • Event mapping table showing auto-mapped events (page_view, cta_click, form_submit, purchase)
  • Back/Next navigation between steps
US-04

Guided Setup Wizard — Verification Step

"As a store owner, I want to know immediately whether my setup worked, with a clear pass/fail result and what to do if something went wrong."
Acceptance Criteria
  • Automated verification checklist with visual pass/fail indicators
  • Each check: ID format validation, tag installation, test event sent, event received
  • Success state: green banner with confirmation message and timeline expectation
  • Failure state: red indicators with plain-language troubleshooting guidance
  • "Finish Setup" returns to Hub; "View Connection Details" navigates to Detail screen
US-05

Connection Detail — Monitoring & Management

"As a marketer, I want to see that my integrations are working properly, monitor event activity, and troubleshoot issues without leaving the platform."
Acceptance Criteria
  • Page header with integration name, connected status, connection date
  • Action buttons: Reconnect, Disconnect
  • KPI cards: Pages Connected, Events Tracked (30d), Last Sync, Uptime
  • Event activity chart showing events over time (Chart.js line chart)
  • Connection details sidebar with technical metadata
  • Connected pages table with per-page event counts and status
  • Recent events log with timestamps
  • Collapsible troubleshooting FAQ section

6. Non-Functional Requirements

  • Design Consistency: Must match the existing Revmatics CRO product aesthetic (same typography, colors, spacing, card styling)
  • Responsiveness: All screens must be desktop-first but responsive to tablet widths
  • Language & Accessibility: Educational content must be written at a reading level accessible to non-technical small business owners
  • Setup Speed: Wizard steps must be completable in under 3 minutes per integration
  • Verification Feedback: Verification step must provide real-time feedback (no "check back later" dead ends)
  • Error Handling: Error states must include plain-language guidance, not technical error codes

7. Success Metrics

Metric Definitions

  • Integration Completion Rate: Percentage of users who start the setup wizard and complete it successfully
  • Time to First Connection: Elapsed time from user's first visit to Integrations Hub to first successfully verified connection
  • Active Integrations per User: Average number of connected integrations per user account
  • Integration Health Score: Percentage of connected integrations with no active errors or warnings over a 30-day period
  • Support Ticket Reduction: Reduction in support tickets with "how do I connect" or "why isn't X working" subject lines

8. Out of Scope

Version 1.0 Exclusions

  • Custom webhook integrations
  • Direct database connections
  • Server-side tracking implementation
  • Multi-account management (one account per integration)
  • Real-time bidding API access
  • Custom event definitions beyond the standard set
  • Integration marketplace or third-party plugin system
